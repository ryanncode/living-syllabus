# Theme Library

This repository includes modified versions of the following open-source projects, adapted for LMS compatibility:

* **Bahunya:** Based on [Bahunya](https://github.com/Kimeiga/bahunya).
* **Latex:** Based on [Latex](https://github.com/vincentdoerig/latex-css/) (MIT License).
* **Marx:** Based on [Marx](https://github.com/mblode/marx/) (MIT License).
* **MCSS:** Based on [MCSS](https://github.com/mikemai2awesome/mcss/) (MIT License).
* **Paper:** Based on [Paper](https://github.com/papercss/papercss/) (ISC License).
* **Pico Green:** Based on [Pico Green](https://github.com/picocss/pico/) (MIT License).
* **Sakura:** Based on [Sakura](https://github.com/oxalorg/sakura/) (MIT License).
* **Simple:** Based on [Simple](https://github.com/kevquirk/simple.css/) (MIT License).
* **Tacit:** Based on [Tacit](https://github.com/yegor256/tacit/) (MIT License).
* **Tufte:** Based on [Tufte](https://github.com/edwardtufte/tufte-css/) (MIT License).
* **Water:** Based on [Water](https://github.com/kognise/water.css/) (MIT License).

All modifications are released under the same license as the original source.
